# Traffic Behavior Simulation

This is the repository for a minimal traffic simulation.

It was developed for a paper that studies the impact that law enforcement has on driver perception and safety. 

## Replication & Requirements

This was developed using Python 3.13

The requirements are installable from ```requirements.txt``` with ```pip install -r requirements.txt```

## How to run

This project can be ran from the ```run_simulation.py``` script with python.
